# PMO-Project-Dashboard
PMO dashboard using Excel 


## 📊 Overview
This is a PMO portfolio project to simulate real-time project tracking.

## 🔧 Tools Used
- Excel


## 📌 Features
- Project tracking (status, deadlines)
- Risk management (severity, mitigation)
- Resource utilization tracking

## 💡 Insights
- Identified delayed tasks
- Monitored resource workload
- Improved reporting visibility
